GitHub Pages + Hugo Tutorial (1)

Hugo Hugo는 Go로 작성된 정적 사이트 생성기(Static Site Generator)로 웹사이트를 빠르고 효율적으로 구축할 수 있게 해주는 도구입니다. Hugo 주요 특징 고성능: Hugo는 Go 언어로 작성되어 있어 매우 빠른 성능을 자랑합니다. 큰 사이트도 몇 초 내에 생성할 수 있습니다. 간편한 설치: 별도의 의존성이나 데이터베이스 없이 단일 실행 파일로 설치되며, 이는 사용과 배포를 매우 간단하게 만듭니다. 유연성과 확장성: Hugo는 다양한 테마와 템플릿을 지원하며, 마크다운(Markdown)을 사용하여 콘텐츠를 작성할 수 있습니다. 실시간 리로딩: 로컬 개발 서버에서 작업 시, 파일 변경을 자동으로 감지하여 실시간으로 웹사이트를 업데이트합니다....
